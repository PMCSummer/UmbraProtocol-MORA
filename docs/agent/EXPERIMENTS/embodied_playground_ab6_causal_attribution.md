# AB6 / Self-World Causal Attribution Integration

## Purpose
AB6 integrates AP01/effect/frontier/update evidence into bounded self/world causal attribution frames without overclaiming self-causation.

## Relation to AB1-AB5
- AB1 provides non-causal event digest evidence.
- AB2/AB3 provide competing explanation structure.
- AB4 provides epistemic basis refs (not action authority).
- AB5 provides support-delta updates after effects.
- AB6 consumes these references for attribution framing.

## Relation to P10/P11/P9
- P10 supplies embodied request/effect lineage evidence.
- P11 remains perturbation battery evidence source.
- P9 strict no-auto-builder constraints remain compatible (public basis only, no hidden substitution).

## Demo cases
- `self_action_correlated_effect`
- `world_only_change`
- `other_actor_change`
- `delayed_self_effect`
- `mixed_self_world_effect`
- `unknown_unexplained_effect`
- `sensor_projection_mismatch`
- `blocked_action_no_success`
- `hidden_eval_only_cause`

## Allowed claims
- MORA can produce bounded self/world causal attribution frames under uncertainty.
- MORA can avoid self-overclaim in world-only and other-actor perturbations.
- MORA can preserve mixed/unknown attribution states.

## Forbidden claims
- full self-model
- true final cause
- complete causal attribution
- consciousness
- human-like agency
- theory of mind

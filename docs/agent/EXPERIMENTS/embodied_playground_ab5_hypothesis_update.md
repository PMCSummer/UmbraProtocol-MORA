# AB5 / Hypothesis Update from Effects

## Purpose
AB5 updates bounded hypothesis support after effect feedback, without turning effects or requests into truth oracles.

## Relation to AB1/AB2/AB3/AB4
- AB1: supplies bounded non-causal event digests.
- AB2: supplies provisional competing hypothesis seeds.
- AB3: supplies explanation frontier (competing hypotheses, unresolved conflicts, missing evidence).
- AB4: supplies bounded epistemic basis refs.
- AB5: updates support from correlated public effects.

## Relation to P10/P11/P9
- P10 provides embodied AP01/effect feedback path evidence.
- P11 constrains ownership overclaim boundaries.
- P9 strict no-auto-builder principles remain preserved (public basis and no hidden/eval substitutions).

## Demo cases
- `correlated_effect_support_increase`
- `disconfirming_effect_support_decrease`
- `ambiguous_effect_no_closure`
- `request_alone_no_confirmation`
- `uncorrelated_effect_weak_or_blocked_update`
- `hidden_eval_effect_rejected`
- `no_effect_no_update`

## Allowed claims
- MORA can update bounded hypothesis support from correlated public effects.
- MORA can disconfirm/weaken hypotheses under contradictory evidence.
- MORA preserves unresolved status under ambiguity.

## Forbidden claims
- final cause confirmed
- anomaly resolved generally
- full abduction
- full active inference
- scientific reasoning
- consciousness
- general reasoning
- epistemic action selection by AB5
